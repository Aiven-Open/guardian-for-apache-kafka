KafkaClient

Companion object KafkaClient

class KafkaClient extends KafkaClientInterface with StrictLogging

A Kafka Client that uses Alpakka Kafka Consumer under the hood to create a stream of events from a Kafka cluster. To configure the Alpakka Kafka Consumer use the standard typesafe configuration i.e. akka.kafka.consumer (note that the keySerializer and valueSerializer are hardcoded so you cannot override this).
